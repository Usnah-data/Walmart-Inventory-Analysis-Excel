#  Walmart Inventory & Sales Analysis (Excel Dashboard)

## Project Overview

This project presents an interactive Excel dashboard analyzing Walmart’s inventory, sales performance, and profitability across product categories and states.

The dashboard provides a clear view of key business metrics and helps identify high-performing products, revenue drivers, and regional profit trends.


##  Business Problem

Retail businesses need to effectively manage inventory while maximizing sales and profit.

This project answers:

* Which product categories generate the most sales?
* What products drive the highest profit?
* How does sales performance trend over time?
* Which states contribute the most to profitability?


##  Key Metrics

* **Total Sales:** $725,457.82
* **Total Profit:** $108,418.45
* **Total Quantity Sold:** 12,264


##  Key Insights

*  **Phones generate the highest revenue**, making them the top-performing category
*  Categories like **Chairs and Storage** contribute significantly to overall sales volume
*  Some categories show **high sales but lower profit margins**, indicating cost inefficiencies
*  **California dominates profit contribution**, followed by Washington
*  Sales show a **steady upward trend over the years**

##  Dashboard Features

* KPI cards (Sales, Profit, Quantity)
* Category-level performance breakdown
* Sales and quantity comparison
* Profit analysis by state
* Yearly sales trend visualization
